ğŸ‘¤ 2. Mengatur Profil

Halaman profil berisi data penting yang akan digunakan di semua dokumen yang Anda buat.

Data yang Perlu Diisi:

  • Nama Lengkap & NIP - Akan muncul di bagian tanda tangan dokumen.
  • Nama Sekolah - Identitas satuan pendidikan.
  • Jenjang - SD/SMP/SMA (mempengaruhi durasi JP default).
  • Kota/Kabupaten - Untuk label tempat pada tanda tangan.
  • Data Kepala Sekolah - Nama dan NIP untuk tanda tangan "Mengetahui".
  • Mata Pelajaran yang Diampu - Bisa lebih dari satu, lengkap dengan JP per minggu.

ğŸ“… 3. Kalender Pendidikan

Kalender pendidikan menentukan kapan semester dimulai/berakhir dan hari-hari libur.

Fitur Kalender:

  • Hari Libur Nasional Tetap - Sudah otomatis terisi (17 Agustus, Natal, dll).
  • Hari Libur Khusus - Anda bisa menambahkan sendiri (Idul Fitri, Idul Adha, dll) karena tanggalnya berubah setiap tahun.
  • Pengaturan Kelas Akhir - Kelas 6/9/12 biasanya selesai lebih awal, bisa diatur terpisah.
  • Ringkasan Minggu Efektif - Sistem menghitung otomatis berapa minggu efektif per semester.

Atur jadwal mengajar Anda dengan fitur validasi anti-bentrok.

Fitur Jadwal:

  • Atur Jam Pelajaran - Custom durasi JP sesuai jenjang (SD 35 menit, SMP 40 menit, SMA 45 menit).
  • Generate Otomatis - Sistem bisa membuat jadwal jam secara otomatis.
  • Validasi Anti-Bentrok - Sistem mencegah guru mengajar di 2 kelas berbeda pada jam yang sama.

ğŸ“„ 5. Generator Dokumen

Ini adalah fitur utama aplikasi. Generate berbagai dokumen dari satu input data.

Dokumen yang Bisa Digenerate:

FREE:
  • ATP (Alur Tujuan Pembelajaran)
  • Prota (Program Tahunan)
PREMIUM:
  • Promes (Program Semester)
  • Modul Ajar
  • KKTP
  • LKPD

Cara Generate:

  1. Upload file CSV berisi CP/TP atau pilih dari data yang tersedia.
  2. Pilih kelas, semester, dan bab yang ingin digenerate.
  3. Centang dokumen yang ingin dibuat.
  4. Klik "Generate Dokumen".
  5. Edit langsung di preview jika perlu (klik pada teks).
  6. Klik "Cetak / PDF" untuk menyimpan.

ğŸ“‹ 6. Format File CSV

File CSV adalah cara termudah untuk memasukkan data CP/TP dalam jumlah banyak.

Format yang Diterima:

Fase;Kelas;Semester;Elemen/Bab;Capaian Pembelajaran;Tujuan Pembelajaran;Dimensi Profil Lulusan

âš ï¸ Penting!

  • Gunakan titik koma (;) sebagai pemisah kolom, BUKAN koma.
  • JANGAN gunakan tanda koma (,) di dalam teks CP dan TP.
  • Tanda koma HANYA boleh di kolom Dimensi Profil Lulusan.
  • Baris pertama adalah header.

Contoh Data:

Fase A;1;Ganjil;Al-Qur'an Hadis / Bab 1;Peserta didik mengenal huruf hijaiyah;Peserta didik mampu melafalkan huruf hijaiyah;Keimanan, Komunikasi

Gunakan AI Assistant untuk membantu membuat file CSV dari materi mentah.

Cara Menggunakan:

  1. Buka menu AI Assistant.
  2. Pilih jenis prompt yang dibutuhkan (CP/TP, Data Siswa, Kalender).
  3. Isi parameter yang diminta.
  4. Klik Generate Prompt.
  5. Copy prompt dan paste ke ChatGPT atau Claude.
  6. Copy hasil AI dan simpan sebagai file CSV.
  7. Upload file CSV ke sistem.

Apakah data saya aman?

Ya, data Anda disimpan di Firebase dengan keamanan tingkat enterprise. Hanya Anda yang bisa mengakses data milik Anda.

Bagaimana cara upgrade ke Premium?

Klik tombol "Upgrade" dan Anda akan diarahkan ke WhatsApp admin untuk melakukan pembayaran.

Bisakah saya menggunakan email selain Gmail?

Tidak. Saat ini sistem hanya mendukung login dengan akun Gmail untuk keamanan dan kemudahan.

Apakah ada versi offline?

Saat ini aplikasi membutuhkan koneksi internet. Namun, dokumen yang sudah digenerate bisa disimpan sebagai PDF untuk akses offline.

Bagaimana jika saya mengajar lebih dari satu mata pelajaran?

Anda bisa menambahkan beberapa mata pelajaran di halaman Profil, masing-masing dengan pengaturan JP per minggu sendiri.

Butuh bantuan lebih lanjut? Hubungi admin melalui WhatsApp.
